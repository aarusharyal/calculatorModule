# Calculator Module

This repository contains a simple calculator module for performing basic mathematical operations.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Length Conversion
- Currency Conversion
- Time Conversion
- Modulo (Coming soon)
- Clear/reset functionality (Coming soon)

---

## Usage

1. Download all files from this repository and save them to a directory.
2. Open a terminal and navigate to that directory using `cd`, or open the terminal directly from the folder.
3. Run the script with:

```bash
node app.js
```

---

## Requirements

- Node.js must be installed to run this code.
